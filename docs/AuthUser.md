账号管理
-------
账户是业务系统中最重要的部分之一。在一个公司中至少拥有一个到多个不同业务系统来做支撑；每个业务系统都会需要不同的业务人员进行操作。此时产生以下特点：
*   给业务人员分配权限比较繁琐
*   业务人员需要记住不同业务系统中的账号密码
*   业务人员离职后，对业务人员的账号清理工作繁重
*   不同业务系统账号管理系统开发成本较高

该系统提供了,
*   **一站式**账号管理，**一次创建，业务畅通无阻**。
*   账号禁用功能，一键解除账号在业务系统的使用权限。
*   **一站式**权限分配
*   不需要重复开发
